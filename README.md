1.Clone the GitHub repository:
    git clone <your-github-repo-link>
2.Open the project using IntelliJ IDEA or Eclipse.
3.Make sure the following are installed:
    Java JDK
    Google Chrome
    ChromeDriver
    Maven 
4.Install Selenium dependencies using Maven by updating the pom.xml file:
        <dependency>
            <groupId>org.seleniumhq.selenium</groupId>
            <artifactId>selenium-java</artifactId>
            <version>4.44.0</version>
            <scope>compile</scope>
        </dependency>
5.Download the ChromeDriver version matching your Chrome browser and configure it if required.
6.Locate the file:
    ContactFormTest.java
7.Run the Java file from the IDE or terminal.
8.Simply click the Run button in IntelliJ/Eclipse.
